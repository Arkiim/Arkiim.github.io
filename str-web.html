Partie 2 : HTML

Structure du site Web

La structure du site est quelque chose de très important, un site non structuré est inconsultable, une catastrophe. C'est donc un des points les plus importants que l'on rencontrera également dans la partie CSS.

Un site est organisé comme suit : le header (le ou les titre(s)), la barre de navigation qui nous permet de nous déplacer dans le site, aussi appelé "navbar".

Représentation de la structure d'un site web [27]

Tout ceci suivi du contenu principal (main content) avec, éventuellement, une ou plusieur barre de navigation ou autre sur les côtés latéraux. Le contenu prinicapal, est constitué de différents articles, paragraphes, photos ...
C'est la partie majeure du site, là où l'essentiel de l'information est encodé.

En dernier lieu, il y a le footer (bas de page) qui sert notamment à contenir les informations complémentaires, tel que le formulaire de contact, plusieurs liens vers les réseaux sociaux et d'autres informations annexes.
