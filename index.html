Introduction

Le Travail de maturité qui va suivre est non pas sous la forme d'un écrit tangible et physique, mais sous la forme d'un site Web.
En quelque sorte, ce site est l'écrit.
Ce site parle de programmation, plus spécifiquement de programmation Web notamment l'HTML et le CSS, il contient quatre parties et se termine par sa conclusion. Voici un "plan" du site et de ses différentes parties :

  1. Page de garde / Introduction
  2. L'Histoire du Web
    1. Notions Fondamentales
    2. W3C
    3. Web 1.0
    4. Web 2.0
    5. Web 3.0
  3. Qu'est-ce que l'HTML ?
    1. Qu’est-ce qu’une balise ?
    2. Composition d’un site
    3. Comment débuter un fichier HTML ?
    4. Comment est structuré le code HTML ?
    5. L’élément <div>
    6. Comment insère-t-on une image ?
    7. Comment insère-t-on un lien ? Qu’est-ce qu’un chemin relatif ?
  4. Qu'est-ce que le CSS ?
  5. Autres
    1. Bibliographies (Autres sources)
    2. Liste des abréviations
  6. Conclusion
    1. Eléments délicats
    2. Remerciements

Toutefois, dans l'optique de réaliser tout ceci, nombres d'autres sujet devront être abordés pour pouvoir expliquer correctement ceux cités ci-dessus. Des sujets tels que la structure d'un site Web, comment fait l'ordinateur pour en afficher un, comment marche un navigateur, qu'est-ce que le langage machine, ou encore, qu'est-ce qu'une balise.
