Partie 4 : Javascript

Qu'est-ce que le JavaScript ?

Le JavaScript est, à l'instar de l'html et du css, le dernier des trois principaux langages de programmation web. Cependant, son usage ne se limite pas à ceci. C'est également l'un des principaux langages, utilisés dans la programmation de fichier .exe, "executables" (programmes).

Logo non-officiel de Javascript [28]

Ce langage s'occupe de toute la partie "interraction" des pages web, par exemple si un site, affiche une alerte du type :

Joyeux 30 ans du Web !!

ou indique la longueur d'une phrase, affiche la date et l'heure en temps réel ( en raffraichissant constamment pour toujours afficher l'heure actuelle ).
L'on peut même faire en sorte qu'après avoir affiché la date que le site souhaite bon anniversaire à l'utilisateur si c'est son anniversaire (et s'il est connu par le programme).

Une autre partie très importante du JavaScript est la réalisation de calul. Il est possible à l'aide du bon code d'effectuer n'importe quel type de calul. C'est pourquoi pour illustrer ceci et montrer de manière plus "flagrante" et "imagée", comment il est possible d'utiliser du JavaScript, une calculatrice a été réalisée.
Néanmoins pour qu'il y ait des intéractions, il faut qu'il y ait de base, des éléments html, structurés par du css.
