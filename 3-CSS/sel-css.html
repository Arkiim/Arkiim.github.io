Partie 3 : CSS

Les sélecteurs

Pour pouvoir appliquer une propriété css à l'élément html sélectionné, il faut que le navigateur trouve aisément la partie que l'on veut styliser, c'est la que les sélecteurs prennent leurs importance. Il existe trois types de sélecteur différent :

  1. Le sélecteur balise, comme ce que l'on a vu juste avant
  2. La classe
  3. L'ID

La classe et l'ID s'utilisent sous forme d'attribut du type <div class="content"> ou <span id="caption">

Il y a une certaine hiérarchie au niveau du css, c'est à dire que l'on peut écrire des déclarations css sans qu'il ne ce passe rien, si le sélecteur de la déclaration fait référence à un élément déjè stylisé. La hiérarchisation se fait dans cet ordre : le sélecteur balise va faire appliquer la propriété css à tout les éléments html du même type que la balise, c'est un sélecteur très général, il est donc tout en bas de la hiérarchie.

La classe est le sélecteur le plus commun, le plus polyvalent, elle doit être ré-inséré dans l'élément html en tant qu'attribut à chaque fois que l'on veut l'appliquer. L'ID est comme la classe mais si et une id et une classe font référence au même élément html, ce seront les propriétés de l'ID qui seront appliqués.
