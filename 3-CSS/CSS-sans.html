Partie 3 : CSS

A quoi sert le CSS ?

Le CSS sert à structurer les pages, à ajouter des "règles" aux éléments html pour leurs donner une forme, une structure. Notamment grâce à outils CSS du type : aligner le texte à droite, au milieu, agrandir les marges, mettre des éléments en colonnes etc...

Le CSS stylise les éléments html, il permet de changer la couleur des éléments, rajouter un background (image de fond), change la font (police d'écriture), aggrandir des éléments, raccourcir des éléments, modifier la taille d'une image, sa position. Le CSS permet de faire toutes ces choses là.

Photo de cette page avec CSS (à gauche) et sans CSS (à droite)

Coder du CSS pourrait être comparé à quelqu'un utilisant libre office qui, en quelque clics, utiliser les mêmes propriétés, couleur, alignement du text... Cependant coder du CSS est beaucoup plus long, en revanche, il permet une liberté beaucoup plus grande, quasi infini et offre un meilleur contrôle.
